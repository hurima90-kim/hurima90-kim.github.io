## MDX란?

MDX는 Markdown과 JSX를 결합한 포맷으로, 문서 안에서 React 컴포넌트를 사용할 수 있습니다.

### 코드 예시

```typescript
const greet = (name: string): string => {
  return `Hello, ${name}!`
}
```
