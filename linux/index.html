📌 리눅스 명령어 & 파일 시스템

  • 현재 위치를 확인하는 pwd (print working directory)

  • 디렉토리를 만드는 mkdir [디렉토리명] (make directory)

  • 디렉토리를 이동하는 cd (change directory)

  • 디렉토리 이동 : cd [디렉토리명]

  • 홈 티렉토리 이동 : cd ~

  • 상위 디렉토리 이동 : cd ../

  • 디렉토리나 파일리스트를 조회하는 ls

  • 현재 위치 기준 디렉터리 조회 : ls

  • 특정 디렉터리 조회 : ls [디렉터리 위치]

  • 숨김 파일 및 파일 사이즈도 함께 조회 : ls -alh

  • 빈 파일 생성 touch [파일명]

  • 디렉토리/파일의 구조의 확인하는 tree

  • 일반 조회 : tree

  • 디렉터리만 조회 : tree -d

  • 숨김파일까지 조회 : tree -a

  • 깊이 지정 조회 : tree -L {깊이}

  • 일치하는 패턴만 조회 : tree -P ‘*.conf’

  • 일치하는 패턴 제외 후 조회 : tree -l ‘*.conf’

  • 디렉토리/파일을 복사하는 cp

  • 파일 복사 : cp [원본 파일] [대상 위치]

  • 다른이름으로 복사 : cp [원본 파일] [위치와 파일명]

  • 디렉터리 복사 : cp -r [원본 위치] [대상 위치]

  • 현재 디렉터리로 복사 : cp -r [원본 위치] .

  • 디렉토리/파일을 이동하는 mv

  • 파일/디렉터리 이동 : mv [원본 위치] [새 위치]

  • 디렉토리/파일을 찾는 find

  • 파일/디렉터리 찾기 : find [찾을 위치] -name [파일/디렉터리명]

  • 디렉토리/파일을 삭제하는 rm

  • 파일 삭제 : rm [삭제할 파일 위치]

  • 디렉터리 삭제 : rm -r [삭제할 디렉터리 위치]

  • 파일 강제 삭제(*주의) : rm -rf [삭제할 파일 위치]
